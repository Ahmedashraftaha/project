Watch Dogs 2

about

Similar to its predecessor, Watch Dogs 2 is an action-adventure game with stealth elements and played from a third-person perspective as Marcus Holloway, a young hacker. ... Alternatively, the player can use the stealth approach, in which they can evade enemies or paralyze them temporarily with Marcus' taser.

System Requirements

  • OS: Windows 7 SP1, Windows 8.1, Windows 10 (64-bit versions only)
  • Processor: Intel Core i5-2400S @ 2.5 GHz / AMD FX 6120 @ 3.5 GHz
  • Memory: 6 GB
  • Graphics: Nvidia GeForce GTX 660 (2GB) / AMD Radeon HD 7870 (2GB) or better
  • Hard Drive: 50 GB available storage
